Metrology procedure guide

Setup

(I) Python scripts (with folder substructure) can be downloaded from here:
Download (version 1.0 from 27.05.2022, latest)

(II) Unzip the archive, creating a folder in known $directory$ (fe. D:\Metrology_v1.0\)
(III) Script is written in Python 3.8 32-bit (but 3.9 and 64-bit versions should work and are faster)
(IV) Necessary Python packages can by installed with pip install pandas matplotlib numpy scipy math statistics

Usage (* denotes possibly automatized actions)

(1)* Copy the output of conoscan run to the \Metrology_vX.Y\data\ subfolder (fe. from folder ...\Conoscan\2022-03-18 13.11\ to folder D:\Metrology_v1.0\data\)

(2)* Main script execute_metrology.py is executed in command prompt by typing python execute_metrology.py

(3) Corner positions will be written to \metrology-versionX-Y\results$DATE-TIME$.txt in form of a csv file
and also printed into terminal right after script finishes (TODO: Format to ITk standard)

(4) Corner procedure visuals will be available in \metrology-versionX-Y\visuals$DATE-TIME$\
